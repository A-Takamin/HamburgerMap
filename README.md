# 東京ハンバーガーMap
東京にあるハンバーガー（グルメバーガーのみ）をマップ化するプロジェクト

## 開発ルール
- 進捗管理：[Projects](https://github.com/A-Takamin/HamburgerMap/projects/2)タブの進捗管理ボードで管理する

## 背景
- 現状、GoogleMapでグルメバーガー屋を探すのが難しい。  
　ジャンクやチェーンがヒットしてしまい、邪魔。  
- ハンバーガーユニットが行ったお店をコレクション化するのが難しい。  
　GoogleMapでは、全てのブックマークが「■」マークになって埋もれてしまう。  
   
→グルメバーガーしか載っていないマップを作成しよう！

## 最終目標
- 誰もが簡単に東京のグルメバーガーを探せる
- それぞれが、好きなグルメバーガーを管理できる

## 要求・要件（ブレインストーミング中
- 登録・ログイン
- マップでグルメバーガー屋の位置が視覚的にわかる（[参考サイト](https://map.mcdonalds.co.jp/)）
- 行ったよ！機能（行ったお店を管理できる
- お気に入り機能
- ★〜★★★★★の評価
- マップのフィルター機能（行ってないお店だけ表示。など）
- 検索機能
- ジャンル分け
- 各店の詳細画面（何を載せる？
